# Summary and FAQ

## Summary
Gorbagana Explorer is a Next.js-based web application for browsing data on the Gorbagana blockchain. It connects to network RPC endpoints to fetch blocks, transactions, and account information, offering developers an interface for inspecting chain activity and diagnosing issues.

## FAQ

### What is Gorbagana Explorer?
Gorbagana Explorer is an open-source web application for exploring data on the Gorbagana blockchain network.

### How do I run the project locally?
Install dependencies using `pnpm install` and start the development server with `pnpm dev`.

### How can I contribute?
Fork the repository, create a pull request with your changes, and ensure all checks pass before submission.

### Where can I report issues or request features?
Use the project's GitHub Issues page to report bugs or suggest new features.
